Persistent filters,
with honest drop behavior.

A medium‑fidelity prototype that fixes the most‑cited bug in Omni·Mart's mobile filter experience: your selections silently vanish when you back out of a product page. This redesign pins them, and when four filters together return zero, the system tells you which one it had to relax and why.

Try it in 30 seconds

  1. 1Tap Filter, pick one option per group, then Apply.
  2. 2If all four together match zero items, one chip relaxes itself and the orange card tells you which and why.
  3. 3Tap a product, then ← back. The filter strip is still there, same order.

What this is fixing

✓
Sticky strip. Today's app drops filters on back‑nav from a product page. The prototype pins them in the same order, scroll position preserved.
!
Honest drop on conflict. Instead of an empty “0 results” page, the least selective filter relaxes, the chip stays visible in dashed grey, and an orange card names the swap.
↻
Recovery is one tap. Tap the greyed chip to swap which filter gets relaxed (counts of items each alternative would bring back), or Clear all to start over.
