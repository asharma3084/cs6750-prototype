# CS 6750 — Individual Project Prototype (CIN4)

Three mid-fidelity mobile design alternatives for the CS 6750 Individual Project, Check-In 4.

> **Academic project.** *Omni-Mart* is a fictional retailer used as an anonymized stand-in for the real e-commerce site studied in needfinding. No affiliation with, or endorsement by, any actual retailer is implied.

**Live demo:** https://asharma3084.github.io/cs6750-prototype/

## Structure

| Path | What it is |
|---|---|
| `index.html` | Landing page with three buttons |
| `alt-a/index.html` | **Comparison Tray** — pin items, side-by-side compare view |
| `alt-b/index.html` | **Seller & Sponsor Labels** — green / blue / orange chips + tap-to-expand tooltip |
| `alt-c/index.html` | **Sticky Filter Strip & Honest Drops** — three-screen static view |
| `alt-c/interactive.html` | Original interactive walkthrough for Alt C |

Each alt page shows three phone-screen mockups side-by-side at identical fidelity, so survey screenshots can be cropped with consistent framing and avoid fidelity bias in side-by-side scoring.

## Survey use

Nine screenshots (three per prototype) are embedded in the CIN3 evaluation survey. Respondents score each prototype on a Likert grid, then forced-rank all three.
