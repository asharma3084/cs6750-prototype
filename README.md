# CS 6750 — Individual Project Prototype

Medium-fidelity interactive prototype for the CS 6750 Individual Project (Check-In 4).

> **Academic project.** *Omni-Mart* is a fictional retailer used as an anonymized stand-in for the real e-commerce site studied in needfinding. No affiliation with, or endorsement by, any actual retailer is implied.

**Live demo:** https://asharma3084.github.io/cs6750-prototype/

Single-file HTML + CSS + JS. Open `index.html` in any modern browser, or use the live link above. Sized for a 390-wide mobile viewport.

## Scope

Omni-Mart kettle search filter UX. Demonstrates:

- Persistent filter chip strip across navigation
- Honest filter drop when constraints conflict (with live recovery count)
- Recovery modal to swap which filter is relaxed
- Clear-all affordance
- Functional bottom navigation (visual scoping only)
